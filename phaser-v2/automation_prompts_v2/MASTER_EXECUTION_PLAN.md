# Master Execution Plan

## Goal
- Turn the Phaser V2 sandbox into a modular Metroidvania engine with strong feel, scalable systems, and support for Shadow Swap, Element Shift, and Weapon Evolution.

## Execution model
- One phase per run.
- Target about 5 minutes per run.
- Implement only the highest-value working slice.
- Stop for manual test after each run.

## Order
1. Foundation and feel
2. Animation and camera impact
3. Input and player state
4. Combat and enemies
5. World and progression
6. Core mechanics integration
7. UI, map, save, content
8. Performance and polish
