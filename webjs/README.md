# Shadow Shift Web

This folder contains a browser-playable prototype of `Shadow Shift`.

How to run:

1. Open `index.html` in a browser

If your browser blocks local script loading, use a tiny local server instead:

- Python: `python -m http.server 8080`
- Then visit: `http://localhost:8080`

Controls:

- `A` / `D`: move
- `Space`: jump
- `Shift`: dash
- `F`: attack
- `E`: swap Light / Shadow world
- `1`: Fire
- `2`: Ice
- `3`: Wind
- `0`: clear element

Prototype slice goals included:

- movement and jump
- dash
- melee attack
- Shadow Swap
- Element Shift
- one enemy
- one Fire barrier
- one Shadow-only platform
- one seal pickup that opens the exit
- one weapon upgrade pickup
