# Legacy JavaScript Archive

This directory stores the deprecated pre-TypeScript Shadow Shift web prototype for historical reference only.

- Archived source: `archive/legacy-js/webjs/game.js` and `archive/legacy-js/webjs/engine/*.js`
- Active production version: `webjs/src/**` (TypeScript + Vite)
- Legacy code in this folder is not part of the active build, run, or deployment path.
